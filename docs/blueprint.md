# **App Name**: PriceWise

## Core Features:

- Image Upload and Product Recognition: Allows users to upload a product image, then uses the Google Vision API tool to identify the product and extract relevant labels.
- Price Comparison: Fetches product prices from eBay and Amazon APIs based on the recognized product, normalizes the results, and displays them in a unified format.
- Search History: Stores and displays the user's last 10 searches, including product name, best price, and search date, allowing users to quickly search again.
- Dashboard Overview: Provides a unified dashboard showing the detected product from the last upload, total price results, the cheapest store, and a search history button.
- User-Friendly Interface: Uses Tailwind CSS for a clean, modern, and responsive design, including well-designed buttons, cards, and modals.

## Style Guidelines:

- Primary color: Deep sky blue (#4169E1), reflecting intelligence, trustworthiness, and technological proficiency, aligning with the application's core functionality.
- Background color: Light blue (#E6EBF5), a desaturated variant of the primary color, offering a gentle and unobtrusive backdrop that enhances content visibility and readability.
- Accent color: Royal blue (#4141E1), slightly different hue and more intense, to ensure primary color contrasts on actionable elements like buttons.
- Headline font: 'Space Grotesk' sans-serif, which imparts a contemporary, computerized aesthetic suitable for headings. Body font: 'Inter' sans-serif, which complements Space Grotesk.
- Simple, geometric icons that clearly represent different product categories and functionalities, ensuring ease of navigation and a visually consistent experience.
- A card-based layout with clear data grids to display search results and history, making information easy to scan and compare. The design must be fully responsive, adapting to different screen sizes.
- Subtle loading animations during image upload and price search processes to provide visual feedback and improve the user experience, indicating that the application is actively working.
